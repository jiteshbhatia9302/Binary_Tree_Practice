Practice Session for Binary Trees

Section: K24FS

Each question carries 5 marks
(a) Create a binary tree of your choice containing 10 nodes. Write functions to display the following traversals of the tree:
  • Inorder traversal
  • Preorder traversal
  • Postorder traversal
Call each function one by one to display the tree traversals. Click here for solution.
(b) Create a binary tree of your choice with 11 nodes, such that 4 nodes are present in the left subtree and 6 nodes are present in the right subtree of the root node. Write a function to display the nodes visible from the left view of the binary tree. Call the function and print the output.
(c) Write a program to check whether two binary trees are mirror images of each other. Test the program using two different test cases, where each binary tree contains more than 8 nodes, such that the test cases produce different results.
(d) Write a program to determine whether a given binary tree is a balanced binary tree or not.
(e) Write a program to find the sum of all nodes in a binary tree.
(f) Write a program to find the maximum value present in a binary tree.
